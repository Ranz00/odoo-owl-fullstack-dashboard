# Odoo OWL Fullstack Dashboard

A technical demonstration of a real-time analytics dashboard integrated into Odoo 17+.

## Technical Stack
- **Backend:** Python & Odoo ORM (PostgreSQL).
- **Frontend:** OWL (Odoo Web Library) & JavaScript ES6+.
- **Communication:** JSON-RPC & REST API patterns.

## Features
- **Reactive UI:** State-managed dashboard components using OWL hooks.
- **Data Aggregation:** Server-side logic for real-time Sales KPIs.
- **Extensibility:** REST controller for external system integration.

## Installation
1. Clone to your Odoo `addons` path.
2. Install `odoo-owl-fullstack-dashboard` from the Apps menu.
3. Access via the **Analytics** menu.
